维修接机登记

这一页不是流程说明，而是正式要用的第一步页面。前提是客户已经同意维修。这个页面只做接机登记：记录客户信息、密码、机器照片、机器号码、以及基础功能检测。

1. 客户信息

姓名必填，联系方式手机号码 / 邮箱二选一即可。
手机号和邮箱至少填一个，后面可以改成提交时校验。

2. 解锁信息

要兼容数字密码、文字密码，以及图案密码。
数字密码
文字 / 混合密码
图案密码
无密码 / 无法提供

3. 机器信息

IMEI、S/N、或无法记录号码，这三种情况都要支持。
IMEI
S/N
无法记录

4. 机器拍照

接机时至少拍正面和背面，边框和现有损伤也建议留图。
正面照片
后面可接相机 / 上传功能
背面照片
左侧边框 / 顶部
右侧边框 / 底部

5. 基础功能检测

能检测就记录，不能检测也要明确选“无法测”。这一页先放基础常用项。
屏幕显示
触摸功能
充电功能
前置 / 后置相机
Face ID / 指纹
开机状态

6. 保存

这一页完成后，再进入后面的维修过程页面。

这一页的目标

只做第一步接机 不讲整个流程，不放订货、不放外发、不放交付。
覆盖真实门店情况 联系方式二选一、密码多种类型、机器号码也不是永远都能记录。
后面方便接数据库 每个模块都已经拆开，后面你接 GitHub 页面和报价系统会更顺。

下一步建议

这页确认后，下一页再做“维修过程页”：

- 是否需要订货
- 主板是否外发
- 发现新问题如何追加报价
- 当前维修状态
