Google Pixel Fingerprint Calibration

Use this after replacing a Google Pixel screen with an under-display fingerprint sensor. The goal is to register the new sensor with Google's repair tool, finish with a manual reboot, then re-enrol the customer's fingerprint.

Pixel 6 and newer only

Use This Guide When

A Pixel 6 or newer has just had the display replaced and fingerprint enrolment is missing, failing, or not available after the repair.

Before You Start

Confirm the screen is fitted correctly, the phone can enter Fastboot mode, and the customer understands the phone will reboot during the process.

1

Prepare The Bench Computer

Use Chrome or Edge on a computer with internet access. On Windows, install Google's USB driver before connecting the phone. On Linux, make sure the user has USB device access and Android platform tools support.

If the bench computer often handles Android repairs, keep the USB driver installed permanently. It prevents most "device not found" failures when the phone is in Fastboot mode.
2

Enable Developer Options Before Powering Off

If the phone still boots into Android, enable Developer options and USB debugging first. Use this normal Android path:

  • Open Settings, then go to About phone.
  • Tap Build number seven times until Developer options are enabled.
  • Go back to System, open Developer options, then turn on USB debugging.
If the phone is already stuck before Android setup, continue with Fastboot mode and test whether the Google tool can still detect it.
3

Open The Pixel Fingerprint Tool

Open the Google Pixel Update and Software Repair fingerprint calibration page on the bench computer. Leave the browser page ready before connecting the phone so the device prompt is easier to catch.

Use the Google tool link in the sidebar.
Allow browser USB permissions when prompted.
Do not use the normal Android file transfer notification for this workflow.
4

Put The Pixel Into Fastboot Mode

Power the phone off. Hold Power and Volume Down together for about three seconds, then release when the Fastboot screen appears.

Connect the phone to the computer with a known good USB-C data cable. Avoid charge-only cables.

The phone should show the Fastboot screen, not normal Android, recovery, or a charging-only screen.
5

Connect And Run Calibration

On the Google repair page, choose the connect device option, select the Pixel from the browser permission popup, then start the software install or calibration step shown by Google.

Wait for the download and install process to finish. The phone may reboot automatically while the tool is applying the calibration package.

Do not disconnect the cable while the browser is downloading, installing, or waiting for the phone to reboot.
6

Finish With Manual Reboot And Test

When the browser reports that the install succeeded, unplug the phone and manually restart it once more. After Android boots, remove any old fingerprint entry if needed and enrol a new fingerprint.

Fingerprint setup screen opens without an error.
New fingerprint can be enrolled from start to finish.
Lock and unlock test works at least three times.
7

Troubleshooting Notes

  • Device does not appear: try another USB port, another data cable, reinstall the Google USB driver, and confirm the phone is in Fastboot mode.
  • Browser cannot connect: use Chrome or Edge, close other Android tools, reload the Google tool page, then reconnect the phone.
  • Download or install fails: repeat the process once from a clean Fastboot start. If the same Google server error repeats, record the exact message with the job notes.
  • Calibration says success but fingerprint still fails: manually reboot again, delete old fingerprints, re-enrol, and inspect the display/sensor connection before replacing parts.
