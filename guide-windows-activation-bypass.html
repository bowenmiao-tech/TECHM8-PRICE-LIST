Windows Activation Setup Bypass

Use this quick internal guide when the machine is stuck on the activation or out-of-box setup screen and you need to open a local-only setup path.

Guide Summary

What This Guide Covers

This is a simplified staff reference. It only includes the two actions needed to open Command Prompt and run the local-only setup command.

1

Open Command Prompt

On the activation screen, press Shift + F10 to open a Command Prompt window (also called CMD).

2

Run the Local Setup Command

Type the command below, then press Enter.

start ms-cxh:localonly
Quick Copy

Command Reference

Use this exact command in the CMD window:
