TECHM8 价格管理后台

这是一个不用数据库的免费版本：你在后台改价后，复制或下载新的 data.json，提交到 GitHub， 前台会通过防缓存请求和定时轮询自动拉取最新价格。也就是说，不是秒级推送，但已经做到部署后自动刷新前台。
GitHub Pages 方案
Step 1
先在这里编辑品牌、型号、维修项目、价格和时间。
Step 2
点击“复制 JSON”或“下载 data.json”，把最新数据准备好。
Step 3
点击“打开 GitHub 编辑页”，粘贴并提交到你的仓库。
Step 4
前台页面会定时检查新的 data.json，GitHub Pages 更新后会自动拉到新价格。
